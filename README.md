Video Downloader

Video Downloader je program za Windows za prenašanje,
pretvarjanje in osnovno urejanje videoposnetkov.

Funkcije
prenos videoposnetkov in podnapisov
čakalna vrsta več prenosov
pretvarjanje videa, zvoka in slik
urejevalnik videoposnetkov
izrez začetka in konca
učinki, prekrivanje in zameglitev
zapečeni in ločeni podnapisi
največ 42 znakov na vrstico
največ dve vrstici oziroma 84 znakov

Prenos
Najnovejša različica je na voljo v razdelku Releases.

Pomembno
Program uporabljajte samo za vsebine, za katere imate dovoljenje
za prenos in uporabo.
Program trenutno ni digitalno podpisan, zato lahko Windows pri prvem
zagonu prikaže varnostno opozorilo.
